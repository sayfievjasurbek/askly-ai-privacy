Privacy Policy

Last Updated: October 2025

TL;DR: We don't collect, store, or transmit any of your personal data. Your API key stays on your device. We don't track you. That's it.

Overview

Explain with AI is committed to protecting your privacy. This privacy policy explains what data we collect (spoiler: nothing), how we use it, and your rights.

What Data We Collect

Data We DO Collect:

None. We do not collect any personal information, browsing data, or usage statistics.

Data Stored Locally on Your Device:

The following data is stored locally in your browser and never leaves your device:

  • OpenAI API Key - Stored so you don't have to re-enter it
  • Extension Settings - Your preferences (backend URL, etc.)

This data is stored using Chrome's built-in storage API and synchronized across your Chrome browsers if you're signed in. This synchronization is handled entirely by Google Chrome, not by us.

How Your Data is Used

Text You Highlight:

When you highlight text and click "Explain with AI":

  1. Selected text is sent directly from your browser to OpenAI's API
  2. OpenAI processes the text and returns an explanation
  3. The explanation is displayed in the extension
  4. We never see, store, or have access to this data

Your API Key:

  • Stored locally in your browser only
  • Never sent to our servers (we don't have servers)
  • Only sent to OpenAI's API when making requests
  • You can delete it anytime from settings

Data Sharing and Third Parties

OpenAI:

When you use the extension, text is sent to OpenAI's API. OpenAI's privacy policy governs how they handle this data:

  • OpenAI Privacy Policy
  • OpenAI API Data Usage

Important: As of March 2023, OpenAI states that data sent via API is not used to train their models unless you explicitly opt in.

No Other Third Parties:

We do not share data with any other third parties. We do not use analytics, tracking, advertising, or any other third-party services.

Nothing to sell

Required Permissions Explained

Chrome requires us to declare permissions. Here's why we need each one:

  • activeTab - Read text when you highlight it and click our button
  • storage - Save your API key locally in your browser
  • contextMenus - Add "Explain with AI" to right-click menu
  • scripting - Detect text selection on webpages
  • <all_urls> - Work on any website (only activates when you use it)

Data Security

Local Storage: Your API key is encrypted by Chrome's built-in security and accessible only by this extension.

Transmission: All connections to OpenAI use HTTPS encryption. Direct from your browser to OpenAI - no intermediate servers.

Your Rights and Choices

You can:

  • ✅ View your stored API key anytime (in extension settings)
  • ✅ Delete your API key anytime (in extension settings)
  • ✅ Uninstall the extension (removes all local data)
  • ✅ Choose not to use the extension on specific websites
  • ✅ Review our source code (open source)

Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect information from children.

International Users

This extension works worldwide. Your data is processed by OpenAI (US-based). OpenAI's privacy policy applies to their data handling.

GDPR & CCPA Compliance

Good news: We don't collect personal data, so there's nothing to delete or opt-out from. Your API key is stored locally - you are the data controller.
